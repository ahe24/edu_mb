# 강의용 슬라이드 사이트 구현 계획서 review

## 2. 사용자 결정 필요 사항 (User Review Required)


### 2.1 메인 프레임워크 선택

- **옵션 B (강력 추천: Slidev or Next.js + Reveal.js)** 로 선택
- 향후 연동할 "데모 목업"의 기술 스택 :  React/Vue 형태의 인터랙티브 컴포넌트.

### 2.2 다중 강의 URL 라우팅 및 서버 환경
강의가 여러 개 등록될 경우 URL을 분리(`edadev.ednc.co.kr/mb/lec01`)하기 위한 배포 아키텍처입니다.

- **결정 필요:** 배포 타겟인 리눅스 배포 서버(Rocky 9) 환경에 Nginx(또는 Apache)와 같은 리버스 프록시 웹 서버가 세팅되어 사용 가능한가요? 
  - No : Node.js(Express) 스크립트로 각 강의 폴더를 호스팅하는 자체 웹 서버 코드를 작성한 뒤, 전체를 PM2로 8900번 포트에서 구동.

### 2.3 테마 및 디자인 커스터마이징
전체 사이트는 기술 내용 전달에 초점을 맞춰 가독성이 높고 편안한 테마로 구성하며, 강의마다 쉽게 메인 색상 테마를 바꿀 수 있도록 CSS Variables를 전역으로 설계할 계획입니다. (다크/라이트 모드 지원 포함)

- 선호하는 대표 테마 색상 : (민트 또는 다크 그린 등)이 있다면 공유해 주세요.
